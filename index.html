Usuarios en línea

Poner una imagen
Sólo debes poner el comando $img:: seguido de la url de tu imagen, ej. 
$img::http://www.micafeina.com/wp-content/uploads/2011/07/Nyan-Cat.png

Poner una link
Sólo debes poner el comando $url:: seguido del link, ej. 
$url::http://google.com

Poner un video
Sólo debes poner el comando $youtube:: seguido del id del video, ej. 
$youtube::QH2-TGUlwu4

Poner un código inline
Sólo debes ecribir el código entre dos pares de corchetes, ej. 
[[alert('soy un codigo');]]

Poner un código block
Sólo debes escribir el código entre las etiquetas [code][/code], ej. 
[code]
<script>
  alert('soy un codigo');
</script>
[/code]

Limpiar chat
Sólo debes de escribir el comando $clear::

Mandar un mensaje en un popup
Sólo debes poner el comando $alert:: seguido del mensaje, ej. 
$alert::soy una alerta bonita :D

Mandar un sonidito
Sólo debes poner el comando $sonidito::

Cambiar de nombre
Sólo debes poner el comando $rename:: seguido de tu nuevo nombre 
$rename::Fuckencio

Enviar mensaje privado
Sólo debes poner el comando $privado:: seguido del nombre destinatario y seguido del mensaje, ej. 
$privado::Fuckencio::hola! :D
